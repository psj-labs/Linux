# VI editor
#### 리눅스에서 VI editor은 터미널에서 키보드 명령과 모드 전환으로 텍스트를 빠르고 효율적으로 편집하는 유닉스 계열 표준 텍스트 편집기입니다


---

## 실행 / 저장 / 종료

| 명령어 | 의미 |
|---|---|
| `vi file` | 파일 열기 또는 생성 |
| `:w` | 저장 |
| `:q` | 종료 (변경 없을 때) |
| `:q!` | 저장하지 않고 강제 종료 |
| `:wq` | 저장 후 종료 |
| `ZZ` | 저장 후 종료 |

---

## 모드 전환 / 입력

| 명령어 | 의미 |
|---|---|
| `i` | 현재 커서 위치에서 입력 |
| `I` | 현재 줄 맨 앞에서 입력 |
| `a` | 커서 뒤에서 입력 |
| `A` | 현재 줄 맨 끝에서 입력 |
| `o` | 다음 줄 생성 후 입력 |
| `O` | 이전 줄 생성 후 입력 |
| `Esc` | Command 모드로 전환 |

---

## 커서 이동

| 명령어 | 의미 |
|---|---|
| `h` | 왼쪽 이동 |
| `j` | 아래 이동 |
| `k` | 위 이동 |
| `l` | 오른쪽 이동 |
| `0` | 줄 맨 앞으로 이동 |
| `^` | 첫 글자로 이동 |
| `$` | 줄 맨 끝으로 이동 |
| `gg` | 파일 맨 위 |
| `G` | 파일 맨 아래 |
| `nG` | n번째 줄로 이동 |

---

## 삭제

| 명령어 | 의미 |
|---|---|
| `x` | 커서 위치 문자 삭제 |
| `dd` | 한 줄 삭제 |
| `ndd` | n줄 삭제 |
| `D` | 커서부터 줄 끝까지 삭제 |

---

## 복사 / 붙여넣기

| 명령어 | 의미 |
|---|---|
| `yy` | 한 줄 복사 |
| `nyy` | n줄 복사 |
| `p` | 아래/뒤에 붙여넣기 |
| `P` | 위/앞에 붙여넣기 |

---

## 실행 취소 / 복구

| 명령어 | 의미 |
|---|---|
| `u` | 실행 취소 |
| `Ctrl + r` | 취소 복구 |

---

## 검색

| 명령어 | 의미 |
|---|---|
| `/word` | 아래 방향 검색 |
| `?word` | 위 방향 검색 |
| `n` | 다음 검색 결과 |
| `N` | 이전 검색 결과 |

---

## 치환 (문자열 변경)

| 명령어 | 의미 |
|---|---|
| `:s/a/b/` | 현재 줄 첫 a → b |
| `:s/a/b/g` | 현재 줄 전체 치환 |
| `:%s/a/b/g` | 전체 문서 치환 |
| `:%s/a/b/gc` | 전체 문서 확인 후 치환 |

---

## 화면 / 옵션 설정

| 명령어 | 의미 |
|---|---|
| `:set nu` | 줄 번호 표시 |
| `:set nonu` | 줄 번호 숨김 |
| `:set ic` | 대소문자 무시 검색 |
| `:set noic` | 대소문자 구분 검색 |
