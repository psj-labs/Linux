# VI Editor

- **vi editor는 리눅스/유닉스 계열에서 기본으로 제공되는 텍스트 편집기로, 키보드 중심의 조작과 모드 기반 동작 방식이 특징입니다.**

---

## vi editor 실행

- vi editor는 다음 명령어로 실행합니다.
```text
vi [파일명]
```
![01](/VI%20Editor/imgs/01.png)

- 파일이 존재하지 않는 경우, vi는 새 파일을 생성합니다.
- 하단에 `"test.txt" [New File]` 이 표시되며, 이는 **새 파일이 생성되었음을 의미**합니다.

![02](/VI%20Editor/imgs/02.png)

## 명령 모드 전환 및 저장/종료

- Insert 모드에서 **명령 모드로 돌아가기 위해 `Esc` 키**를 누릅니다.
- 명령 모드에서 사용할 수 있는 기본 명령은 다음과 같습니다.

  - `:wq` → 저장 후 종료
  - `:q!` → 저장하지 않고 강제 종료

- 필자는 저장 후 종료를 위해 `:wq`를 입력했습니다.

![05](/VI%20Editor/imgs/05.png)

## 줄 번호 표시

- 화면 좌측에 줄 번호를 표시하면 가독성이 향상됩니다.
- 명령 모드에서 다음 명령을 입력합니다.
```text
:set nu
```
![09](/VI%20Editor/imgs/09.png)

## 문자열 치환 (검색 및 변경)

- vi editor에서는 특정 문자열을 다른 문자열로 치환할 수 있습니다.
- 명령 모드에서 다음 형식의 명령어를 사용합니다.
```text
:%s/원래문자열/바꿀문자열/옵션
```
- 옵션 설명:
  - `g` → 문서 전체에서 전역 치환
  - `i` → 대소문자 구분 없이 치환
  - `c` → 치환 여부를 하나씩 확인

![11](/VI%20Editor/imgs/11.png)

- 필자는 `i` 문자열을 `a`로 전역 치환하기 위해 다음 명령을 사용했습니다.
```text
:%s/i/a/g
```
- 총 3개의 라인이 치환되었다는 메시지와 함께 치환이 완료되었습니다.

![12](/VI%20Editor/imgs/12.png)

## 문자열 검색

- 특정 문자열을 검색하려면 명령 모드에서 `/`를 사용합니다.
- 검색 형식은 다음과 같습니다.
```text
/[검색할문자열]
```
- 필자는 `s` 문자열을 검색했습니다.

![13](/VI%20Editor/imgs/13.png)

- 동일한 문자열이 여러 곳에 존재할 경우, `n` 키를 눌러 다음 검색 결과로 이동할 수 있습니다.

![14](/VI%20Editor/imgs/14.png)

---

## 정리

- vi editor는 **모드 기반 편집기**로, 명령 모드와 Insert 모드를 구분해 사용합니다.
- 기본적인 파일 편집, 저장, 검색, 치환 기능만 익혀도 리눅스 환경에서 효율적인 작업이 가능합니다.
